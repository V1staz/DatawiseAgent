## 3.3 Docker 镜像

DatawiseAgent 默认使用 Docker sandbox 执行 notebook 代码。

```bash
  . --progress=plain
```

- [ ] 构建：

```bash

  -f datawiseagent/coding/jupyter/cuda_jupyter_server.dockerfile \
  . --progress=plain
```

备注：

- DataModeling 如果要尽量贴近论文，建议准备 GPU 镜像。
- 但当前配置文件有一处 bug，见下文“必须调整项”。

## P0-2 `configs/datamodeling.yaml` 的 GPU 镜像配置位置错误

文件：`configs/datamodeling.yaml`

问题：

- `image_name: my-jupyter-image-gpus` 当前被写在 `agent.plan` 下
- 实际执行器读取的是 `code_executor.image_name`

处理建议（二选一）：

- [ ] 方案 A：直接修 YAML，把 `image_name` 放到 `code_executor:` 下
- [ ] 方案 B：创建 session 时手工传 `reset_code_executor`

推荐：

- [ ] 用方案 A，最直观
